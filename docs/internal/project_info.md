# Información del proyecto

¡Bienvenido al equipo! Esta es aplicación web para la gestión de un colegio canino, incluyendo
clientes, mascotas, matrículas y asistencia.

## Stack Tecnológico

- **Backend:** Python, Django, Django REST Framework
- **Frontend:** React, Vite, TypeScript, Material UI
- **Base de Datos:** PostgreSQL
- **Alojamiento:** Render
